Сборка программы:
    1. ./configure_cmake.sh     (конфигурация сборки - при первой установке проекта)
    2. ./linux_build.sh         (сборка)

Запуск программы:
    Вариант 1: через скрипт - ./run.sh
    Вариант 2: через исполняемый файл ./build/bin/sfml-app
